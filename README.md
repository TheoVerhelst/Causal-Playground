# Causal Playground

Implementations of core algorithms and models in causality.

This repository is a small playground for experimenting with fundamental concepts in causal inference, including graphical models, conditional independence, d-separation, and counterfactual reasoning.

## Run the examples

    python -m examples.example_counterfactuals

## Run the tests

    python -m test.test_d_separation

## Repository purpose

The goal of this repository is to provide simple implementations of basic causal tools and ideas, with an emphasis on clarity and experimentation rather than large-scale application code.

## Contents

The repository includes implementations related to:

- Causal graphs;
- D-separation;
- Counterfactual reasoning;
- Basic causal models and algorithms.

## Notes

This repository is intended as a lightweight research and experimentation codebase for fundamental causality methods.
